# electrode-native-website

The [Docusaurus][1]-powered website and documentation for [Electrode Native][2].

## Getting Started

### Install dependencies

```sh
yarn
```

### Run the dev server:

```sh
yarn start
```

## Full Documentation

Full documentation can be found on the [website](https://docusaurus.io/).
